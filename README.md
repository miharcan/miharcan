## Featured Projects

### 1) grounded-react
A retrieval-grounded ReAct agent framework that reasons strictly from documentation and source code.

- Why it matters: improves factuality by constraining reasoning to evidence
- Stack: Python, retrieval pipelines, agent orchestration
- Repo: https://github.com/miharcan/grounded-react

### 2) llm-platform
Multi-tenant RAG platform with JWT tenant isolation, RBAC, Qdrant filtering, and PostgreSQL audit logging.

- Why it matters: practical blueprint for secure multi-tenant AI services
- Stack: Python, FastAPI, Qdrant, PostgreSQL
- Repo: https://github.com/miharcan/llm-platform

### 3) chatbot_guardrails
Guardrail API for detecting personalized mental-health advice with explainable safe/unsafe decisions.

- Why it matters: safety layer for high-risk AI interactions
- Stack: FastAPI, Docker, policy checks, explainability
- Repo: https://github.com/miharcan/chatbot_guardrails

### 4) event-driven-alpha
Leakage-controlled framework for structured/semantic news signals in time-series return prediction.

- Why it matters: rigorous walk-forward validation for signal research
- Stack: Python, NLP, time-series modeling
- Repo: https://github.com/miharcan/event-driven-alpha
